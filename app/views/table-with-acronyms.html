What do these acronyms mean?

  • RFP = Request for Proposal (responding to client brief)
  • SOW = Statement of Work (drafting project scope)
  • KO = Kickoff (initial project meeting)
  • WIP = Work in Progress (active development)
  • UAT = User Acceptance Testing (client testing phase)
  • IMPL = Implementation (deploying to production)
  • BAU = Business as Usual (ongoing support)
  • HOLD = On Hold (paused, awaiting client decision)
  • TBD = To Be Determined (not yet scheduled)

If everyone needs this explanation, why use acronyms at all? Writing out the full terms benefits everyone—not just screen reader users.
